## 🎯 متطلبات التصميم

### دليل الألوان الرسمي

```css
:root {
  --edham-primary: #DC2626;      /* أحمر إدهام */
  --edham-dark: #1A1A1A;         /* أسود داكن */
  --edham-gray: #2D2D2D;         /* رمادي */
  --edham-white: #FFFFFF;        /* أبيض */
}
```

### متطلبات الجودة

- **الدقة:** 72 DPI للويب، 300 DPI للطباعة
- **الصيغة:** JPG للصور الفوتوغرافية، PNG للشعارات، SVG للأيقونات
- **الحجم:** لا يتجاوز 500KB للصور الويب
- **النسبة:** 16:9 للهيدر، 4:3 للمعرض، 1:1 للشعارات

## 💡 ملاحظات مهمة للمصور/المصمم

1. **التوجيهات الفنية:**
   - جميع صور الشاحنات يجب أن تكون من الجانب أو الزاوية الأمامية
   - خلفية نظيفة أو مبهمة (bokeh) للتركيز على الشاحنة
   - إضاءة طبيعية أو استوديو احترافي
   - عدم وجود علامات مائية أو حقوق محفوظة لجهات أخرى

2. **المحتوى الممنوع:**
   - لا صور مخالفة للأنظمة المرورية
   - لا لوحات تسجيل واضحة (يمكن تمويهها)
   - لا أشخاص بدون إذن استخدام الصورة

3. **التفضيلات:**
   - شاحنات حديثة (2020+)
   - شاحنات بيضاء أو حمراء تتوافق مع هوية العلامة
   - بيئة سعودية/عربية (لافتات عربية، أعلام)
