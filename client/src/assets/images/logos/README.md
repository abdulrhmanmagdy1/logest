# 🎨 **شعارات نظام إدهام اللوجستي**

## 📋 **نظرة عامة**

مجموعة شعارات احترافية ومتعددة الاستخدامات لنظام إدهام اللوجستي، مصممة لتعكس الاحترافية والثقة في الخدمات اللوجستية.

---

## 🟢 **الشعار الرئيسي (Primary Logo)**

### **📄 الوصف**
الشعار الرئيسي لنظام إدهام، يجمع بين هوية العلامة التجارية والطابع اللوجستي الحديث.

### **🎨 التصميم**
- **الألوان الأساسية:**
  - الأزرق الداكن: `#1e3a8a` (للثقة والاحترافية)
  - البرتقالي الزاهي: `#f97316` (للنشاط والحركة)
  - الأبيض: `#ffffff` (للنقاء والوضوح)

### **📐 المكونات**
1. **رمز إدهام:** شاحنة لوجستية مصممة بأسلوب عصري
2. **اسم النظام:** "إدهام" بخط عربي أنيق
3. **الشعار:** يدمج بين الحروف العربية والرمز اللوجستي

### **📱 الاستخدامات**
- الموقع الإلكتروني
- تطبيقات الموبايل
- الوثائق والتقارير
- البطاقات والزي الرسمي

---

## 📐 **متغيرات الشعارات (Logo Variations)**

### **🔄 التنسيقات المدعومة**
- **PNG:** للويب والتطبيقات الرقمية
- **SVG:** للواجهات التفاعلية والرسوم المتحركة
- **ICO:** لأيقونات التطبيقات
- **PDF:** عالي الجودة للطباعة
- **AI:** قابلة للتعديل بالذكاء الاصطناعي

### **📏 الأحجام المدعومة**
- **16x16:** للأيقونات الصغيرة
- **32x32:** للتطبيقات
- **64x64:** للويب
- **128x128:** للعروض التقديمية
- **256x256:** للطباعة عالية الجودة
- **512x512:** للعروض الكبيرة
- **1024x1024:** للشاشات عالية الدقة
- **SVG:** متغير الحجم (responsive)

## 🎯 **إرشادات الاستخدام**

### **🌐 للموقع الإلكتروني**
```html

<img src="/assets/logos/edham-primary.svg" alt="إدهام لوجستي" class="logo-primary">

<!-- الشعار البرتقالي للواجهات التشغيلية -->
<img src="/assets/logos/edham-orange.svg" alt="إدهام لوجستي" class="logo-orange">

<!-- الشعار الأخضر للواجهات التسجيل -->
<img src="/assets/logos/edham-green.svg" alt="إدهام لوجستي" class="logo-green">
```

### **📱 لتطبيقات الموبايل**
```swift
// iOS (Swift)
struct EdhamLogo {
    static let primary = "edham-primary"
    static let orange = "edham-orange" 
    static let green = "edham-green"
}
```

```kotlin
// Android (Kotlin)
object EdhamLogos {
    const val PRIMARY = "edham-primary"
    const val ORANGE = "edham-orange"
    const val GREEN = "edham-green"
}
```

### **📄 للطباعة والوثائق**
```css
/* للطباعة */
.logo-print {
    color: #000000;
}

/* للوثائق الرسمية */
.logo-official {
    color: #1e3a8a;
}
```
