# 🚀 نظام EDHAM لللوجستيات المتكاملة
## Edham Unified Logistics System

---

## 📋 نظرة عامة

**EDHAM** هو نظام لوجستي ذكي متكامل يوفر حلولاً شاملة لإدارة العمليات اللوجستية بكفاءة عالية. تم تصميم النظام لخدمة مختلف الأدوار في شركة النقل واللوجستيات، من العملاء والسائقين إلى المشرفين والمحاسبين وفرق الصيانة.

### 🎯 الرؤية
توفير منصة لوجستية ذكية، موحدة، وسهلة الاستخدام، تربط بين جميع الأطراف المعنية في سلسلة الإمداد، مع التركيز على:
- ⚡ الكفاءة والسرعة
- 📊 الشفافية والتتبع
- 🤖 الأتمتة والذكاء الاصطناعي
- 🔒 الأمان والموثوقية

---

## 🏗️ الميزات الرئيسية

### 1️⃣ تطبيق موحد متعدد الأدوار
تطبيق واحد يخدم **5 أدوار** مختلفة:

| الدور | الوظيفة | المميزات |
|-------|---------|----------|
| 👤 **العميل** | طلب ومتباعة الشحنات | طلب سهل، تتبع live، فواتير |
| 🚛 **السائق** | تنفيذ الرحلات | GPS، رفع إثباتات، استبيان |
| 🧑‍💼 **المشرف** | إدارة العمليات | تعيين رحلات، خريطة live، تنبيهات |
| 💰 **المحاسب** | إدارة المالية | فواتير، سندات قبض، تقارير |
| 🔧 **الصيانة** | صيانة الأسطول | تسجيل صيانة، مخزون قطع |

### 2️⃣ خريطة تتبع مباشرة (Live Map)
- 📍 تتبع GPS لجميع الشاحنات في الوقت الفعلي
- 🗺️ عرض الشحنات حسب الحالة:
  - 🟢 **شحنات مكتملة** (تم التسليم)
  - 🟡 **شحنات قيد التنفيذ** (في الطريق)
  - 🔵 **شحنات جديدة** (في انتظار التعيين)
- 📊 إحصائيات مباشرة على الخريطة

### 3️⃣ نظام ذكي متكامل (AI-Powered)
- 🤖 **المساعد الذكي**: روبوت محادثة للمساعدة
- 📈 **التنبؤ بالطلب**: تحليلات ذكية للتخطيط
- ⚠️ **كشف الشذوذ**: إنذارات مبكرة للمشاكل
- 🛣️ **توصية المسارات**: أمثل مسار تلقائياً

### 4️⃣ نظام متعدد المشرفين
- 👥 إمكانية وجود **أكثر من مشرف**
- 🔐 صلاحيات مرنة لكل مشرف
- 📊 لوحة تحكم مركزية للإدارة
- 🔔 نظام تنبيهات ذكي

### 5️⃣ رسوم بيانية وتحليلات متقدمة
- 📈 رسوم بيانية تفاعلية
- 📊 لوحات معلومات (Dashboards)
- 📉 تقارير أداء مفصلة
- 📅 تقارير دورية تلقائية

## 🛠️ التقنيات المستخدمة

### Backend (الخادم)
```
✅ Node.js + Express.js
✅ MongoDB (قاعدة بيانات NoSQL)
✅ Redis (التخزين المؤقت)
✅ Socket.io (الإتصال الفوري)
✅ JWT (المصادقة)
✅ Stripe (المدفوعات)
```

### Mobile (الموبايل)
```
✅ Kotlin (لغة البرمجة)
✅ Jetpack Compose (واجهة المستخدم)
✅ Hilt (حقن التبعيات)
✅ Google Maps SDK (الخرائط)
✅ Fused Location (تحديد الموقع)
```

### الذكاء الاصطناعي والتكامل
```
✅ AI/ML للتنبؤ والتحليل
✅ Blockchain للعقود الذكية
✅ IoT لمراقبة المركبات
✅ EDI لتبادل البيانات
```

## 🚀 نشر النظام

### خطوات النشر:
```bash
# 1. نسخ الكود
git clone https://github.com/edham/logistics.git

# 2. تثبيت التبعيات
npm install

# 3. إعداد البيئة
cp .env.example .env

# 4. تشغيل الـ Backend
npm start

# 5. بناء تطبيق Android
```
