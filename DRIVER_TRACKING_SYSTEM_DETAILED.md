## 🔄 عمليات تحديث الموقع (Location Update Operations)

### **1️⃣ تحديث الموقع المستمر (Continuous Location Updates)**

**التكرار:**
```
- المرة الأولى: كل 5 ثواني (عند النشاط الكامل)
- المرة العادية: كل 10 ثواني (أثناء الرحلة)
- المرة الثالثة: كل 30 ثانية (في الخلفية)
- موفر البطارية: كل 60 ثانية (وضع توفير الطاقة)
```

**البيانات المرسلة:**
```javascript
PUT /api/v1/shipments/:id/status

{
  "status": "on_the_way",
  "location": {
    "lat": 24.8500,
    "lng": 46.7200,
    "altitude": 456,
    "accuracy": 15,
    "speed": 85,
    "heading": 45,
    "timestamp": "2026-05-22T12:45:30Z"
  },
  "vehicle": {
    "fuel": 0.65,
    "temperature": 22,
    "status": "good"
  },
  "notes": "في الطريق بسلام"
}
```

**الاستجابة من الخادم:**
```json
{
  "success": true,
  "message": "تم تحديث الموقع",
  "data": {
    "tripId": "507f1f77bcf86cd799439011",
    "status": "on_the_way",
    "lastUpdate": "2026-05-22T12:45:30Z",
    "distanceRemaining": 45,
    "eta": "2026-05-22T13:30:00Z"
  }
}
```

### **2️⃣ تحديث الحالة (Status Update)**

**الحالات الممكنة:**
```
at_pickup      → في موقع الاستلام
picked_up      → تم استلام البضاعة
on_the_way     → في الطريق
at_delivery    → في موقع التسليم
delivered      → تم التسليم
completed      → انتهت الرحلة
```

**مثال: تحديث إلى "picked_up":**
```javascript
PUT /api/v1/shipments/:id/status

{
  "status": "picked_up",
  "location": {
    "lat": 24.7136,
    "lng": 46.6753,
    "timestamp": "2026-05-22T09:30:00Z"
  },
  "signature": "base64_encoded_image_data",
  "photo": "base64_encoded_image_data",
  "notes": "تم التقاط الصورة والتوقيع"
}
```
