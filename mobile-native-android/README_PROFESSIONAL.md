# 🚀 إدهام اللوجستي - التطبيق الاحترافي

## 📋 نظرة عامة

تطبيق "إدهام اللوجستي" هو نظام لوجستي احترافي متكامل من الجيل الرابع، مصمم بأحدث التقنيات وأفضل الممارسات البرمجية. يتميز التطبيق بواجهة مستخدم فاخرة مع هوية بصرية نيونية موحدة، وتقنيات متقدمة تشمل الذكاء الاصطناعي، الواقع المعزز، والبلوك تشين.

## 🎯 الميزات الرئيسية

### 🎨 **الهوية البصرية الاحترافية**
- **تصميم داكن نيوني** مع اللون البرتقالي الداكن
- **تأثيرات حركية سلسة** وانتقالات احترافية
- **تصميم زجاجي (Glass)** وشفافيات متقدمة
- **تأثيرات إضاءة متوهجة** ديناميكية

### 🤖 **التقنيات المتقدمة**
- **الذكاء الاصطناعي (AI)** للتنبؤات والتحسينات
- **الواقع المعزز (AR)** للقياسات والملاحة
- **البلوك تشين (Blockchain)** للتوثيق والأمان
- **الأداء المحسّن** مع تخزين مؤقت ذكي

### 👥 **أدوار المستخدمين**
- **العميل الذكي** - لوحة تحكم متقدمة مع AI و AR
- **السائق المحترف** - تتبع مباشر وإدارة الشحنات
- **المشرف المتميز** - إدارة النظام والتحليلات
- **فني الصيانة** - إدارة المعدات والصيانة الوقائية
- **المحاسب الدقيق** - التقارير المالية والفواتير

## 🚀 **المكونات الاحترافية**

### **ProfessionalAnimations.kt**
- تأثيرات حركية متقدمة (Slide, Scale, Fade)
- انتقالات سلسة بين الشاشات
- تأثيرات نيون متوهجة

- أدوات مساعدة للواجهة
- مؤشرات حالة متحركة
- حالات تحميل فارغة وخطأ

## 📱 **التثبيت والتشغيل**

### **متطلبات النظام**
- **Android** - API 24 (Android 7.0) أو أعلى
- **iOS** - iOS 12.0 أو أعلى
- **Web** - متصفح حديث مع دعم JavaScript

### **خطوات التثبيت**
1. **استنساخ المستودع**
   ```bash
   git clone https://github.com/edham/logistics-app.git
   cd logistics-app
   ```

2. **تثبيت تطبيق الأندرويد**
   ```bash
   cd mobile-native-android
   ./gradlew assembleDebug
   ```

3. **تثبيت تطبيق الويب**
   ```bash
   cd client
   npm install
   npm start
   ```

4. **تشغيل الخادم الخلفي**
   ```bash
   cd backend
   npm install
   npm run dev
   ```
