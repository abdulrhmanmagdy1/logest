# نظام إدهام للخدمات اللوجستية (Edham Logistics System) 🚛

**تطوير:** فريق Aws Digital

نظام لوجستي متكامل يهدف إلى تحسين إدارة الأسطول، تتبع المركبات، وتيسير عمليات التسليم من خلال حلول تقنية مبتكرة لتعزيز الشفافية والكفاءة.

---

## 🎯 أهداف النظام الرئيسية
*   **إدارة الحمولات:** نظام شامل لإدارة الشاحنات بفعالية وسرعة.
*   **تحديد الموقع:** تتبع تلقائي لموقع السائق لضمان تنفيذ المهام في الوقت المحدد.
*   **توثيق العمليات:** تسجيل دقيق للحمولات وسندات القبض لتعزيز المصداقية.
*   **الإدارة المالية:** إصدار فواتير دقيقة وإدارة المديونيات لتفادي الأخطاء المالية.
*   **تقييم الأداء:** نظام استبيان لجمع آراء السائقين وتحسين الخدمة.

---

## 🛣 المسارات الرئيسية للنظام (User Roles)
1.  **العميل (Customer):** طلب الحمولات ومتابعة الحالة.
2.  **المشرف (Supervisor):** إدارة الإرساليات ومتابعة تنفيذ المهام.
3.  **المحاسب (Accountant):** إصدار الفواتير وسندات القبض.
4.  **السائق (Driver):** تحديث الموقع، تسجيل الرحلات، ورفع المرفقات.
5.  **الورشة والصيانة (Workshop):** توثيق الصيانة، تتبع تغيير الزيت، وإدارة قطع الغيار.

---

## 🛠 التقنيات المستخدمة (Tech Stack)
*   **Android:** Native Kotlin (Clean Architecture, MVVM, Hilt, Jetpack Compose).
*   **iOS:** Native Swift (لنسخة الـ iOS).
*   **Backend:** Integrated API System.
*   **Database:** آمنة ومشفرة مع بوابات دفع متكاملة.

---

## 📂 هيكلية السورس كود (Source Code Structure)
*   `/app`: تطبيق الأندرويد (Kotlin).

*   `/docs`: التقارير الفنية والوثائق.
