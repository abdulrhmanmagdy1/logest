# خارطة الطريق الاحترافية - iOS Native (Swift)

هذا الملف هو مرجع التنفيذ الرسمي لنسخة الأيفون "إدهام" لضمان التطابق الكامل مع نسخة الأندرويد.

## الهدف
بناء تطبيق لوجستي iOS Native باستخدام **SwiftUI** و **Combine**، يتميز بالأداء العالي، الأمان، وسهولة الصيانة.

## حالة التنفيذ الحالية
- [x] **Sprint 0 - الأساس المعماري الأولي**
  - إعداد هيكل المشروع (Core, UI, Network)
  - إعداد `APIService` و `SessionManager`
  - إعداد نظام الألوان (Cockpit Theme)

- [x] **Sprint 1 - المصادقة والجلسات (Auth & Session)**
  - بناء شاشة الدخول (Login View)
  - إدارة الجلسة باستخدام Keychain/UserDefaults
  - نظام التوجيه التلقائي حسب الدور

- [x] **Sprint 2 - لوحة تحكم السائق والعميل (Dashboards)**
  - بناء واجهات الـ Cockpit للسائق والعميل والمشرف.
  - إضافة نظام التنبيهات الحية (Local Notifications).

- [x] **Sprint 3 - إدارة الشحنات وإثبات التسليم (POD)**
  - قائمة وتفاصيل الشحنات.
  - إثبات التسليم (التوقيع الرقمي + تصوير المستندات).

- [x] **Sprint 4 - التتبع والخرائط (GPS & Maps)**
  - تكامل CoreLocation و MapKit.
  - عرض المسارات المباشرة للسائق والعميل.

- [x] **Sprint 5 - المشرف والتحليلات المتقدمة (Supervisor & Charts)**
  - بناء لوحة تحكم المشرف المتقدمة.
  - عرض الرسوم البيانية (Swift Charts) للحرارة والأداء.
  - نظام مكافآت السائقين وتتبع الأداء.

## الخطوات القادمة
1. **الصيانة والوقود:** إضافة قسم تسجيل مصاريف الوقود والصيانة للسائق.
2. **الدردشة الحية:** إضافة نظام دردشة فورية بين السائق والمشرف.
3. **الوضع غير المتصل:** تحسين العمل في المناطق ضعيفة التغطية.
