# خارطة الطريق الاحترافية - Native Android

هذا الملف هو مرجع التنفيذ الرسمي لمشروع أندرويد "إدهام" بخطوات تدريجية ومنظمة.

## الهدف

بناء تطبيق لوجستي Native Android قابل للتوسع، سهل الصيانة، ومرن للإنتاج بدون كسر الميزات الحالية.

## النطاق الأساسي (حسب المتطلبات)

1. المصادقة
2. الأدوار والصلاحيات
3. لوحة التحكم
4. دورة الشحنة
5. تتبع السائق
6. الإشعارات
7. الفوترة
8. الصيانة

## قواعد الجودة

- كل Feature جديدة تعتمد نمط: `data / domain / presentation`
- لا نضيف شاشة جديدة بدون حالات: Loading / Empty / Error
- لا نكسر API الحالية أثناء الترتيب
- كل Sprint ينتهي بـ Build ناجح وتوثيق واضح
