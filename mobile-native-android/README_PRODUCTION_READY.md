# 🚛 تطبيق إدهام اللوجستي - إصدار الإنتاج الجاهز

## 📋 نظرة عامة

تطبيق إدهام اللوجستي هو نظام متكامل لإدارة النقل المبرد والشحنات والسائقين والصيانة والمحاسبة والتقارير. التطبيق مصمم ليخدم 6 أدوار مختلفة مع واجهات متخصصة لكل دور.

## 🏗️ الهيكلية المعمارية

### الأدوار المدعومة
- **ADMIN** - المدير: وصول كامل لجميع الوظائف
- **SUPERVISOR** - المشرف: إدارة العمليات والأسطول
- **DRIVER** - السائق: إدارة المهام والتتبع
- **CUSTOMER** - العميل: طلب الشحنات والمتابعة
- **ACCOUNTANT** - المحاسب: إدارة الفواتير والمدفوعات
- **WORKSHOP** - الورشة: إدارة الصيانة والمركبات

### المكونات الرئيسية

#### 📱 واجهات المستخدم
- **AdminDashboardFragment** - لوحة تحكم المدير المتقدمة
- **AdvancedAnalyticsFragment** - التحليلات المتقدمة والرسوم البيانية
- **FleetManagementFragment** - إدارة الأسطول التفاعلية
- **LoadAdapter** - محول عرض الشحنات
- **DriverAdapter** - محول عرض السائقين
- **VehicleAdapter** - محول عرض المركبات

#### 🔧 الخدمات المتقدمة
- **SmartNotificationsService** - نظام الإشعارات الذكي
- **LocationTrackingService** - خدمة تتبع المواقع
- **DataSyncService** - خدمة مزامنة البيانات

#### 📊 نماذج البيانات
- **Load** - نموذج الشحنة
- **Driver** - نموذج السائق
- **Vehicle** - نموذج المركبة
- **Invoice** - نموذج الفاتورة
- **MaintenanceRecord** - نموذج سجل الصيانة
- **Notification** - نموذج الإشعار

## 🚀 الإعداد والتشغيل

### المتطلبات الأساسية
- **Android Studio Arctic Fox** أو أحدث
- **Android SDK 30** أو أحدث
- **Kotlin 1.5+**
- **Google Maps API Key**

### خطوات التثبيت
1. **استنساخ المشروع**
   ```bash
   git clone [repository-url]
   cd mobile-native-android
   ```

2. **إعداد API Keys**
   - إضافة Google Maps API key في `local.properties`
   - `MAPS_API_KEY=your_api_key_here`

3. **تشغيل التطبيق**
   - فتح المشروع في Android Studio
   - تشغيل `./gradlew assembleDebug`
   - تثبيت الـ APK على الجهاز

### إعدادات الإنتاج
1. **توقيع التطبيق**
   - إنشاء keystore للتوقيع
   - إعداد signing config في `build.gradle`

2. **تحسين الحجم**
   - تشغيل R8 shrinking
   - تحسين الموارد
   - ضغط الصور

3. **النشر**
   - إنشاء release APK/AAB
   - رفع على Google Play Console
   - إعداد قائمة التطبيق
