# Edham Logistics Platform Frontend

هذه المساحة تم إعدادها كبداية لبناء منصة لوجستية حديثة تحت اسم `Edham Logistics Platform` باستخدام:

- `Next.js` + `TypeScript`
- `Tailwind CSS`
- `Framer Motion`
- `React Query`
- `Zustand`
- `Socket.io` (عند التكامل مع الباك إند)

## الهدف

بناء واجهة قوية تدعم:

- لوحة تحكم احترافية
- نظام تسجيل دخول وتسجيل مستخدمين
- تتبع مباشر للخدمات اللوجستية
- إدارة الشحنات والأسطول
- تجربة مستخدم مميزة وسهلة
- دعم ثيم داكن وفاتح

## هذه البنية تبدأ من:

- `src/app/` للمسارات الأساسية
- `src/components/` للمكونات القابلة لإعادة الاستخدام
- `src/lib/` لطبقات التفاعل مع API
- `src/types/` لأنواع TypeScript
- `src/styles/` لأنماط المشروع
- `src/data/` للبيانات الوهمية (Dummy Data)
